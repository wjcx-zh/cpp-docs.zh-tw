# <a name="atl-wizards-and-dialog-boxes"></a>ATL 嚮導和對話方塊

Active Template Library （ATL）的嚮導會針對各種類型的 COM 物件產生重複的程式碼。 您可以在**方案總管**中開啟專案的快捷方式功能表，然後選擇 [**加入**]、[**類別**]，來執行這些嚮導。
