# <a name="largest-array-size"></a>陣列大小上限

**ANSI 3.3.3.4、4.1.1**：保留陣列大小上限所需的整數型別，即 **size_t** 的大小

`size_t`Typedef 是 **`unsigned int`** 32 位 x86 平臺上的。 在64位平臺上， `size_t` typedef 為 **`unsigned __int64`** 。
