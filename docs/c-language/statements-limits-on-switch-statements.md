# <a name="statements-limits-on-switch-statements"></a>陳述式：Switch 陳述式的限制

**ANSI 3.6.4.2**語句中的最大 **`case`** 值數目 **`switch`**

Microsoft C 不會限制 **`case`** 語句中的值數目 **`switch`** 。 這些數目只會受到可用記憶體的限制。
