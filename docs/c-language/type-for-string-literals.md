# <a name="type-for-string-literals"></a>字串常值的類型

字串常值具有的類型陣列 **`char`** （也就是**char []**）。 （寬字元字串具有的 **`wchar_t`** 類型陣列（也就是**wchar_t []**））。這表示字串是包含類型之元素的陣列 **`char`** 。 陣列中的元素數目等於字串中的字元數目加一 (代表終止 Null 字元)。
