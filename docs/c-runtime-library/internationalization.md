# <a name="internationalization"></a>國際化

Microsoft 執行時間程式庫提供許多常式，讓您可以用來自訂您的應用程式以用於國際市場，例如：
- [地區設定相關常式](../c-runtime-library/locale.md)
- 寬字元常式
- 多位元組字元常式
- 泛型文字常式

為了方便起見，大部分的地區設定相關常式也會依其用途分類。

多位元組常式和寬字元常式會在其存在時，連同其單一位元組字元的對應專案一起描述。

也包含 ISO646 操作員替代專案。
