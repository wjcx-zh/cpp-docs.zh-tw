# <a name="ceil-ceilf-ceill"></a>ceil、ceilf、ceill

計算最大值。

## <a name="syntax"></a>語法

```C
double ceil(
   double x
);
float ceil(
   float x
);  // C++ only
long double ceil(
   long double x
);  // C++ only
float ceilf(
   float x
);
long double ceill(
   long double x
);
```

### <a name="parameters"></a>參數

*x*<br/>
浮點值。

## <a name="return-value"></a>傳回值

**Ceil**函式會傳回浮點值，表示大於或等於*x*的最小整數。 不會傳回錯誤。

|輸入|SEH 例外狀況|Matherr 例外狀況|
|-----------|-------------------|-----------------------|
|± **QNAN**， **IND**|無|**_DOMAIN**|

**ceil**具有使用 Streaming SIMD Extensions 2 （SSE2）的執行。 如需使用 SSE2 實作的資訊和限制，請參閱 [_set_SSE2_enable](set-sse2-enable.md)。

## <a name="remarks"></a>備註

因為 c + + 允許多載，所以您可以呼叫採用或類型之**ceil**的多載 **`float`** **`long double`** 。 在 C 程式中， **ceil**一律會採用並傳回 **`double`** 。

根據預設，此函式的全域狀態範圍設定為應用程式。 若要變更此項，請參閱[CRT 中的全域狀態](../global-state.md)。

## <a name="requirements"></a>需求

|-------------|---------------------|
|**ceil**、 **ceilf**、 **ceill**|\<math.h>|

如需其他相容性資訊，請參閱 [相容性](../../c-runtime-library/compatibility.md)。
