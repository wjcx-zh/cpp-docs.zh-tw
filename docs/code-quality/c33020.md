# <a name="c33020"></a>C33020

> 警告 C33020：偵測到可能是不正確的 HRESULT 使用方式。

這是高度信賴的警告，表示 HRESULT 傳回的函式會傳回 FALSE 或 false。

## <a name="example"></a>範例

```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return FALSE; // C33020
}
```

使用適當的 HRESULT 值更正這些警告：
```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return E_FAIL; // OK
}
```
