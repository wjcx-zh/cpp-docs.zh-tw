# <a name="_bstr_toperator-"></a>_bstr_t::operator !

**Microsoft 特定的**

檢查封裝的 `BSTR` 是否為 Null 字串。

## <a name="syntax"></a>語法

```
bool operator!( ) const throw( );
```

## <a name="return-value"></a>傳回值

如果是，則會傳回 **`true`** ，否則會傳回 **`false`** 。

**結束 Microsoft 專有**
