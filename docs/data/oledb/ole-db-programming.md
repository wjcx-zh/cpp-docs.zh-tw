# <a name="ole-db-programming"></a>OLE DB 程式設計

OLE DB 範本是 C++ 範本，可透過提供實作許多常用 OLE DB 介面的類別，讓您更輕鬆地使用高效能的 OLE DB 資料庫技術。 此範本庫分為消費者範本和提供者範本。

Visual C++ 也支援精靈建立 OLE DB 起始應用程式。

此外，您也可以使用屬性來執行 OLE DB 的取用者範本。
