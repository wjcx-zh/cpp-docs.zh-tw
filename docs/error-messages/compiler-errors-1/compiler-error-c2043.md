# <a name="compiler-error-c2043"></a>編譯器錯誤 C2043

break 的使用不合法

[Break](../../cpp/break-statement-cpp.md)只有在 **`do`** 、 **`for`** 、 **`while`** 或語句內才合法 **`switch`** 。
