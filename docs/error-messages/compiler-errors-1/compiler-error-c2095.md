# <a name="compiler-error-c2095"></a>編譯器錯誤 C2095

' function '：實際參數的類型為 ' void '：參數 ' number '

傳遞給函式的參數是類型 **`void`** ，這是不允許的。 請改用 void （）的指標 `void *` 。

`number`指出哪一個參數為 **`void`** 。
