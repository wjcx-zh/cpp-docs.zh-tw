# <a name="compiler-error-c2152"></a>編譯器錯誤 C2152

' identifier '：具有不同屬性之函式的指標

具有一個呼叫慣例（、或）之函式的指標 **`__cdecl`** **`__stdcall`** **`__fastcall`** ，會指派給具有另一個呼叫慣例的函式指標。
