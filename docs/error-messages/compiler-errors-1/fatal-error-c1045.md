# <a name="fatal-error-c1045"></a>嚴重錯誤 C1045

編譯器限制 : 連結規格巢狀結構太深

巢狀的外部超過編譯器限制。 外部連結類型允許使用嵌套的外部連結，例如 **`extern`** "c + +"。 降低巢狀外部的數目以解決此錯誤。
