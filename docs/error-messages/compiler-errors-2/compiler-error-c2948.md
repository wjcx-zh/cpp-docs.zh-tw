# <a name="compiler-error-c2948"></a>編譯器錯誤 C2948

明確具現化;特製化上不允許儲存類別規範 ' 規範 '

您無法在先前明確具現化的樣板類別的特製化中使用儲存類別指定名稱（例如 **`extern`** ）。
