# <a name="compiler-error-c3166"></a>編譯器錯誤 C3166

> ' 指標 '：無法將內部 __gc 指標的指標宣告為 ' type ' 的成員

編譯器發現不正確指標宣告（指標的 **`__nogc`** 指標 **`__gc`** ）。

C3166 只能使用過時的編譯器選項來連接 **`/clr:oldSyntax`** 。
