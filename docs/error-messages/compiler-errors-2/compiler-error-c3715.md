# <a name="compiler-error-c3715"></a>編譯器錯誤 C3715

' 指標 '：必須是 ' class ' 的指標

您在或中指定的指標 [`__hook`](../../cpp/hook.md) [`__unhook`](../../cpp/unhook.md) 並未指向有效的類別。 若要修正此錯誤，請確定您的 **`__hook`** 和 **`__unhook`** 呼叫指定了有效類別的指標。
