# <a name="compiler-error-c3817"></a>編譯器錯誤 C3817

' 宣告 '：屬性只能套用至函數

**`property`** 關鍵字只能是套用至函式定義的。

C3817 只能使用過時的編譯器選項 **/clr： oldSyntax**來連線。
