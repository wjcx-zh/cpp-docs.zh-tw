# <a name="compiler-error-c3820"></a>編譯器錯誤 C3820

> '*name*'：初始化運算式必須是受控的

## <a name="remarks"></a>備註

Managed 類型的 *name* 初始設定式也必須是 Managed。 若要更正這個錯誤，請指定 managed 初始化運算式方法。
