# <a name="compiler-warning-level-1-c4034"></a>編譯器警告（層級1） C4034

sizeof 傳回0

**`sizeof`** 運算子會套用至大小為零的運算元（空結構、聯集、類別或列舉類型或類型 **`void`** ）。
