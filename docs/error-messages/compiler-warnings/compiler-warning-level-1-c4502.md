# <a name="compiler-warning-level-1-c4502"></a>編譯器警告 (層級 1) C4502

' 連結規格 ' 需要使用關鍵字 ' extern '，且必須在所有其他規範之前

指定了沒有關鍵詞的連結 **`extern`** 。 連結與非外部類型無關。

編譯器會假設 **`extern`** 關鍵字。
