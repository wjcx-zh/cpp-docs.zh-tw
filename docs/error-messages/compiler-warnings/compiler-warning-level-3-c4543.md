# <a name="compiler-warning-level-3-c4543"></a>編譯器警告 (層級 3) C4543

由屬性 ' no_injected_text ' 隱藏的插入文字

[No_injected_text](../../windows/attributes/no-injected-text.md)的屬性會出現在原始程式碼中，這表示編譯器會防止屬性插入程式碼。

這項警告是為了提醒您，屬性將無法插入程式碼。
