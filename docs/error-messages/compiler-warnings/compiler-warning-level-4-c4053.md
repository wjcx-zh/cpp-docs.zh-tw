# <a name="compiler-warning-level-4-c4053"></a>編譯器警告 (層級 4) C4053

'?:' 有一類型為 void 的運算元

`?:`運算子會提供類型為的運算式 **`void`** 。 運算元的值 **`void`** 未定義。
