# <a name="compiler-warning-level-4-c4132"></a>編譯器警告 (層級 4) C4132

'object' : 應初始化常數物件

常數並未初始化。 具有屬性的符號值無法在 **`const`** 其定義之後變更，應提供初始化值給它。
