# <a name="making-owner-drawn-header-controls"></a>建立主控描繪標題控制項

您可以將標題控制項（[CHeaderCtrl](reference/cheaderctrl-class.md)）的個別專案定義為主控描繪專案。 如需詳細資訊，請參閱 Windows SDK 中的主控描繪[標題控制項](/windows/win32/Controls/header-controls)。
