# <a name="ole-mfc"></a>OLE (MFC)

在您的程式中執行 OLE 功能會以數種方式影響您的使用者介面：

- 視覺編輯（就地啟用）會在程式的視窗中顯示另一個程式的使用者介面，並使用其他程式中的專案來修改程式的功能表。

- 拖放功能可讓使用者在視窗內和之間拖曳物件，甚至是在程式之間。

- 追蹤器會在視覺編輯和拖放期間，將視覺提示提供給物件的狀態。
