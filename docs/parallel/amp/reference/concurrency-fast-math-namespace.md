# <a name="concurrencyfast_math-namespace"></a>Concurrency::fast_math 命名空間

命名空間中的函式 `fast_math` 具有較低的精確度、僅支援單精確度（ **`float`** ），並呼叫 DirectX 內建函式。 每個函數都有兩個版本，例如 `cos` 和 `cosf` 。 這兩個版本都採用並傳回 **`float`** ，但每個都會呼叫相同的 DirectX 內建函式。

## <a name="syntax"></a>語法

```cpp
namespace fast_math;
```
