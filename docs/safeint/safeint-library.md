# <a name="safeint-library"></a>SafeInt 程式庫

**SafeInt** 是可以與 MSVC、GCC 或 Clang 搭配使用的可攜式程式庫，有助於防止在應用程式執行數學運算時可能產生的整數溢位。 此程式庫的最新版本位於 [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt) 。
