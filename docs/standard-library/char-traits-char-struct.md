# <a name="char_traitsltchargt-struct"></a>char_traits&lt;char&gt; 結構

結構，這是樣板結構的特製化**char_traits \<CharType> **至類型的專案 **`char`** 。

## <a name="syntax"></a>語法

```cpp
template <>
struct char_traits<char>;
```

## <a name="remarks"></a>備註

特製化可讓結構利用操作此類型物件的程式庫函數 **`char`** 。
