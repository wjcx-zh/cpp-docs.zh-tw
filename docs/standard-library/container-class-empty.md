# <a name="container-classempty"></a>容器類別::empty

> [!NOTE]
> 本主題位於 Microsoft c + + 檔中，做為 c + + 標準程式庫中使用之容器的無作用範例。 如需詳細資訊，請參閱 [C++ 標準程式庫容器](../standard-library/stl-containers.md)。

傳回 **`true`** 空的受控制序列。

## <a name="syntax"></a>語法

```cpp
bool empty() const;
```
