# <a name="ltstackgt"></a>&lt;stack&gt;

定義類別範本堆疊和兩個支援的範本。

## <a name="requirements"></a>規格需求

**標頭：**\<stack>

**命名空間：** std

> [!NOTE]
> 連結 \<stack> 庫也會使用 `#include <initializer_list>` 語句。
