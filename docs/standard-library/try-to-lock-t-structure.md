# <a name="try_to_lock_t-structure"></a>try_to_lock_t 結構

表示 **`struct`** 定義[try_to_lock](../standard-library/mutex-functions.md#try_to_lock)物件的。 用來選取 [unique_lock](../standard-library/unique-lock-class.md) 的其中一個多載建構函式。

## <a name="syntax"></a>語法

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>需求

**標頭：**\<mutex>

**命名空間：** std
