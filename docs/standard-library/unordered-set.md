# <a name="ltunordered_setgt"></a>&lt;unordered_set&gt;

## <a name="requirements"></a>規格需求

**標頭：**\<unordered_set>

**命名空間：** std

> [!NOTE]
> 連結 \<unordered_set> 庫也會使用 `#include <initializer_list>` 語句。
