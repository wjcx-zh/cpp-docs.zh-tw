# <a name="requestedit"></a>requestedit

表示屬性支援 `OnRequestEdit` 通知。

## <a name="syntax"></a>語法

```cpp
[requestedit]
```

## <a name="remarks"></a>備註

**Requestedit** c + + 屬性具有與[requestedit](/windows/win32/Midl/requestedit) MIDL 屬性相同的功能。

## <a name="example"></a>範例

[如需](bindable.md) **requestedit**的範例用法，請參閱可系結的範例。

## <a name="requirements"></a>規格需求

| 屬性內容 | 值 |
|-|-|
|**適用於**|介面方法|
|**重複**|否|
|**必要的屬性**|無|
|**無效屬性**|無|
