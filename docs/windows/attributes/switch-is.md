# <a name="switch_is"></a>switch_is

指定運算式或識別碼，做為選取聯集成員的聯集判別。

## <a name="syntax"></a>語法

```cpp
[switch_is]
```

## <a name="remarks"></a>備註

**Switch_is** c + + 屬性具有與[switch_is](/windows/win32/Midl/switch-is) MIDL 屬性相同的功能。
